#Wuzzuf Job Scraper

A Python project that scrapes job listings from Wuzzuf and saves the data for analysis. Ideal for job market research, salary insights, and building datasets for data science projects.

##Tools & Libraries

Python – Programming language used for development.

Requests – For fetching webpage content.

BeautifulSoup – For parsing HTML and extracting job information.

Pandas – For cleaning, structuring, and exporting data.
